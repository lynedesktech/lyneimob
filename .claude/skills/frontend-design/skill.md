# Skill: Frontend Design

## Objetivo
Garantir que toda interface criada neste projeto seja visualmente profissional,
consistente e de alta qualidade. Esta skill define os padrões visuais a seguir.

## Quando usar
Use esta skill OBRIGATORIAMENTE sempre que for:
- Criar uma nova página ou tela
- Criar ou modificar componentes visuais
- Alterar layout, cores, tipografia ou espaçamento
- Trabalhar com CSS, Tailwind ou qualquer estilo

## Princípios de design
1. **Consistência** — use os mesmos padrões de cor, espaçamento e tipografia em todo o projeto
2. **Simplicidade** — interfaces limpas e sem excesso de elementos
3. **Hierarquia visual** — o que é mais importante deve se destacar mais
4. **Responsividade** — funcionar bem em diferentes tamanhos de tela
5. **Acessibilidade** — contraste adequado, textos legíveis, elementos clicáveis com tamanho suficiente

## Antes de implementar qualquer interface
1. Verifique se já existe um componente parecido no projeto — reutilize
2. Siga os padrões de cor e tipografia definidos acima
3. Teste visualmente antes de considerar pronto
4. Nunca misture estilos diferentes no mesmo projeto
